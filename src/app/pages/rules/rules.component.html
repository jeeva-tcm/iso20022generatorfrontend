03
Business Rules & Reference Data
Semantic & SWIFT Compliance

Applies semantic-level validation, cross-field business rules, and reference data verification (BIC/Currency) based on SWIFT CBPR+ guidelines.

  • check_circle Purpose Code validation against ISO Code Sets (SALA/RENT/SUPP)
  • check_circle BIC code verification (SWIFT Directory) & Currency code validation (ISO 4217)
  • check_circle Charge Bearer validation (SLEV, SHAR, CRED, DEBT) logic
  • check_circle Conditional Amount Limits and High-Value payment flagging
  • gpp_bad Sanctions Screening — Fail-fast country & party name screening against sanctioned jurisdictions (OFAC / UN / EU lists); any match raises a SANCTIONS_BLOCKED error and aborts processing
